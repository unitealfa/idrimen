# idrimen

## Config
- Cree un fichier `.env` avec `FINNHUB_API_KEY=<ta_cle>`.  
- Si tu utilises un token sandbox (`sandbox_...`), il sera detecte automatiquement. Tu peux aussi forcer le sandbox avec `FINNHUB_USE_SANDBOX=true`.
- Lance le serveur : `npm install` puis `npm start`, et ouvre `http://localhost:3000`.
